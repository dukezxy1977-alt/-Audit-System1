## 🔧 故障排查

### 问题 1：无法启动应用
**症状**：运行 `python run.py` 出错

**解决方案**：
```bash
# 1. 手动创建虚拟环境
python3 -m venv venv

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
python app.py
```

### 问题 2：依赖安装失败
**症状**：pip install 出错

**解决方案**：
```bash
# 1. 升级 pip
pip install --upgrade pip

# 2. 清除缓存
pip cache purge

# 3. 重新安装
pip install -r requirements.txt -v
```

### 问题 3：文件上传失败
**症状**：选择文件后显示错误

**解决方案**：
- 检查文件格式（必须是 PDF/DOCX/DOC/TXT）
- 检查文件大小（最大 50MB）
- 检查文件是否损坏

### 问题 4：评估没有结果
**症状**：点击上传后一直加载

**解决方案**：
- 打开浏览器开发者工具（F12）
- 查看 Console 标签有无错误
- 查看 Network 标签请求状态
- 检查 Flask 应用是否正常运行
