# 招标投标风险评估系统 - 项目完成总结

## 📋 项目概述

**系统名称**：招标投标风险评估系统 v1.0  
**系统定位**：自动化招标合规性检测与风险评估工具  
**开发语言**：Python  
**前端框架**：HTML5 + CSS3 + JavaScript  
**后端框架**：Flask

- ✅ **rules_engine.py** - 规则引擎（45项指标）

## 🏗️ 系统架构

```
招标风险评估系统
├── 前端层
│   ├── HTML5 界面
│   ├── CSS3 样式
│   └── JavaScript 交互
├── 应用层 (Flask)
│   ├── /api/assess (评估接口)

│   ├── /api/statistics (统计接口)

├── 业务层
│   ├── RiskRules (规则引擎)
│   ├── FileProcessor (文件处理)
│   └── ReportGenerator (报告生成)
└── 数据层
    ├── 上传文件 (uploads/)
    └── 生成报告 (reports/)
```

## 🔄 数据流程

```
用户上传文件
     ↓
FileProcessor 提取文本
     ↓
RiskRules 分析风险
     ↓
计算评分和统计
     ↓
ReportGenerator 生成报告
     ↓
返回结果给前端
     ↓
用户查看和下载报告
```

## 🚀 部署方式

### 本地部署（推荐）
```bash
python run.py  # 自动创建虚拟环境、安装依赖、启动应用
```

### 手动部署
```bash
# 1. 创建虚拟环境
python -m venv venv

# 2. 激活虚拟环境
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate      # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
python app.py

# 5. 访问应用
打开浏览器访问 http://127.0.0.1:5000
```

### 生产部署
可使用以下方式部署到服务器：
- Gunicorn + Nginx
- Docker 容器
- 云平台 (AWS, Azure, 腾讯云等)
