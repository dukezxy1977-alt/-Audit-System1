# 招投标行为风险规则库

## 版本信息
- **版本号**：1.0
- **更新时间**：2024 年
- **依据法规**：招标投标领域公平竞争审查规则（2024 年 5 月 1 日施行）

---

## 第一部分：风险分类体系

### 一、风险等级划分

| 等级 | 描述 | 处理方式 | 影响 |
|------|------|---------|------|
| **A 级（重大违规）** | 违反法律明确禁止规定 | 立即纠正、追责 | 可能导致政策措施撤销、相关人员处分 |
| **B 级（严重违规）** | 存在明显违反公平竞争原则 | 限期改正 | 可能被监督部门纠正 |
| **C 级（中等风险）** | 存在可能限制竞争的因素 | 重点监测 | 可能被投诉异议 |
| **D 级（低风险）** | 程序或规范性问题 | 改进完善 | 需要优化但影响不大 |

---

## 第二部分：具体风险指标

### 模块一：招标人自主权风险

#### 指标 1.1：招标代理机构指定风险 [A 级]
**规范要求**：尊重和保障招标人选择招标代理机构的自主权

**风险表现**：
- 政策措施为招标人指定特定招标代理机构
- 限定招标人选择招标代理机构的具体方式
- 规定强制要求某类招标代理机构资质

**检查清单**：
- [ ] 是否存在"必须选择"某招标代理机构的表述
- [ ] 是否存在"只能通过"特定途径选择的要求
- [ ] 是否存在对代理机构的不合理限制

**应对措施**：
```
如发现违规：
1. 立即删除相关限制性条款
2. 发布修订文件澄清
3. 记录责任人员和纠正过程
```

#### 指标 1.2：投标资格条件指定风险 [A 级]
**规范要求**：不得为招标人指定投标资格、技术、商务条件

**风险表现**：
- 政策强制规定投标企业必须具备的特定资质
- 规定投标企业必须具有特定的技术指标
- 要求投标报价必须在某范围内

**检查清单**：
- [ ] 招标文件中的资格条件是否由招标人自主制定
- [ ] 是否存在政策措施对资格条件的直接指定
- [ ] 条件设置是否与项目需求相适应

**应对措施**：
```
如发现违规：
1. 确认资格条件制定权属于招标人
2. 删除政策中的直接指定条款
3. 修订为"招标人可根据项目需求制定"
```

#### 指标 1.3：评标方法指定风险 [A 级]
**规范要求**：不得为招标人指定资格审查方法或评标方法

**风险表现**：
- 政策要求所有项目必须采用某种评标方法
- 规定某类项目只能使用特定的评分方式
- 强制要求采用比例、质量等特定评标方法

**检查清单**：
- [ ] 评标方法选择权是否完全由招标人行使
- [ ] 政策文件中是否含有"必须采用"的表述
- [ ] 是否为不同类型项目提供多种评标方法选项

**应对措施**：
```
修订方向：
"招标人可根据项目特点选择合理的评标方法，
可采用经评审的最低价法、综合评分法等"
```

#### 指标 1.4：评标标准指定风险 [A 级]
**规范要求**：不得为招标人指定具体的评标标准

**风险表现**：
- 政策规定评标标准必须包含的具体项目
- 规定评分权重固定不变
- 要求必须采用某种评分算法

**检查清单**：
- [ ] 评标标准是否由招标人独立制定
- [ ] 政策中是否涉及评分权重规定
- [ ] 标准灵活性是否满足不同项目需求

**应对措施**：
```
改进方向：
提供示范文本供参考，但明确说明
"招标人可根据项目特点制定，不受示范文本限制"
```

#### 指标 1.5：评标委员会成员指定风险 [A 级]
**规范要求**：不得为招标人指定评标委员会成员

**风险表现**：
- 政策要求评标委员会成员从特定库中抽取
- 规定评标委员会成员必须具备特定身份
- 要求必须聘请政府部门的人员参加评标

**检查清单**：
- [ ] 评标委员会成员由谁组建（招标人还是政府部门）
- [ ] 是否存在"评标委员会成员必须由...组成"的规定
- [ ] 抽取程序是否尊重招标人自主权

**应对措施**：
```
规范做法：
"评标委员会由招标人按招标投标法规定自行组建，
可从专家库中抽取"
```

#### 指标 1.6：电子交易系统选择限制风险 [A 级]
**规范要求**：不得限制已纳入统一平台的电子交易系统选择

**风险表现**：
- 规定所有项目必须在某个特定平台进行交易
- 禁止使用其他合规的交易系统
- 要求项目必须采用本地公共资源交易平台

**检查清单**：
- [ ] 是否涉及公共资源交易平台垄断的规定
- [ ] 其他合规平台是否被排斥
- [ ] 企业选择合规平台的权利是否被保障

**应对措施**：
```
修订建议：
"可使用已纳入统一平台体系的任何合规电子系统"
```

#### 指标 1.7：电子认证服务强制风险 [A 级]
**规范要求**：不得强制选择电子认证服务

**风险表现**：
- 政策规定投标必须使用特定的电子认证服务
- 要求必须从指定机构购买电子认证服务
- 禁止使用其他形式的身份认证

**检查清单**：
- [ ] 是否存在"必须使用...电子认证"的要求
- [ ] 认证服务机构是否为唯一指定
- [ ] 是否有收费情况且对企业造成负担

**应对措施**：
```
改进方向：
取消强制性要求，改为"可选用合规的认证服务"
```

#### 指标 1.8：交易工具指定风险 [A 级]
**规范要求**：不得为招标人指定特定交易工具

**风险表现**：
- 规定投标必须使用特定的报价工具或系统
- 强制要求采用特定的合同订立工具
- 规定保证金必须通过特定工具提交

**检查清单**：
- [ ] 是否规定了唯一的交易工具
- [ ] 多种工具是否同时可用
- [ ] 企业是否有选择的自由

**应对措施**：
```
合规表述：
"可使用合规的交易工具，如...、...等"
```

#### 指标 1.9：承包商预选库限制风险 [A 级]
**规范要求**：不得指定承包商预选库

**风险表现**：
- 规定投标企业必须进入某预选库
- 只允许预选库内的企业投标
- 设置进入预选库的不合理条件

**检查清单**：
- [ ] 是否存在强制预选库规定
- [ ] 进入条件是否合理
- [ ] 预选库之外的企业是否被排斥

**应对措施**：
```
改进方向：
取消强制预选库制度，改为公开竞争招标
```

#### 指标 1.10：地方扶持政策强制执行风险 [A 级]
**规范要求**：不得要求招标人依照地方扶持政策开展招标

**风险表现**：
- 强制要求项目优先采购创新产品
- 规定项目必须优先聘请本地企业
- 要求项目采购本地产品

**检查清单**：
- [ ] 是否存在"必须优先"某类产品或企业的规定
- [ ] 是否与国家统一的市场准入相悖
- [ ] 是否限制了招标人的采购自主权

**应对措施**：
```
改进方向：
改为"可鼓励采购...，但不作强制要求"
```

#### 指标 1.11：其他自主权限制风险 [B 级]
**规范要求**：不得用其他不合理条件限制招标人自主权

**风险表现**：
- 要求招标人进行特定形式的变更
- 限制招标人变更的权利
- 不合理的招标公告内容要求

**检查清单**：
- [ ] 是否存在"招标人必须..."的不合理强制条款
- [ ] 是否充分尊重招标人的权益
- [ ] 条款是否过度干涉招标人自主权

**应对措施**：
```
梳理所有对招标人的要求，
确认均为推荐性而非强制性
```

### 模块二：市场准入与投标资格风险

#### 指标 2.1：市场准入行政许可风险 [A 级]
**规范要求**：不得要求市场准入负面清单以外的行业取得行政许可

**风险表现**：
- 对于非许可行业的企业，政策仍要求取得许可
- 要求投标企业办理特定的行政许可
- 未经许可的企业被禁止参加投标

**检查清单**：
- [ ] 是否涉及许可要求的政策
- [ ] 许可是否在市场准入负面清单内
- [ ] 是否存在超出清单范围的许可要求

**应对措施**：
```
审查步骤：
1. 查阅最新市场准入负面清单
2. 对比政策中的许可要求
3. 删除清单外的许可要求
4. 发布澄清公告
```

#### 指标 2.2：本地分支机构要求风险 [A 级]
**规范要求**：不得要求企业在本地区设立分支机构

**风险表现**：
- 政策要求投标企业必须在本地设有办公室
- 规定没有本地分支机构的企业不能投标
- 要求投标前必须完成分支机构注册

**检查清单**：
- [ ] 是否存在本地分支机构要求
- [ ] 是否对外地企业设置了不合理障碍
- [ ] 该要求是否与项目需求无关

**应对措施**：
```
删除相关条款，改为：
"招标不要求企业在本地设立分支机构"
```

#### 指标 2.3：本地税收社保缴纳要求风险 [A 级]
**规范要求**：不得要求企业缴纳本地税收社保

**风险表现**：
- 规定参加投标企业必须在本地缴纳税款
- 要求企业在本地缴纳社会保险
- 以税收社保缴纳记录作为投标资格

**检查清单**：
- [ ] 投标资格条件中是否包含税收社保要求
- [ ] 是否对外地企业造成歧视
- [ ] 该要求是否为项目必需

**应对措施**：
```
清单化纠正：
1. 删除招标条件中的税收社保要求
2. 若有特殊需求，经公平竞争审查后说明
3. 向相关企业澄清
```

#### 指标 2.4：本地业绩奖项要求风险 [A 级]
**规范要求**：不得要求企业取得本地区业绩或奖项

**风险表现**：
- 规定投标企业必须有本地完成的工程业绩
- 要求企业获得本地颁发的奖项
- 限制外地企业因无本地业绩而不能投标

**检查清单**：
- [ ] 招标文件中是否涉及本地业绩要求
- [ ] 是否存在"仅认可本地业绩"的表述
- [ ] 外地企业是否被等同对待

**应对措施**：
```
修订建议：
改为"认可国内外同类业绩"
或删除业绩要求，改为资质条件
```

#### 指标 2.5：地方证书强制要求风险 [A 级]
**规范要求**：不得要求取得特定地区或行业组织颁发的证书

**风险表现**：
- 规定投标企业必须取得本地行业协会颁发的证书
- 要求企业参加本地组织的培训并获证
- 限制没有本地上岗证的企业投标

**检查清单**：
- [ ] 是否存在地方或行业组织颁发证书的强制要求
- [ ] 是否存在"仅认可本地证书"的规定
- [ ] 是否排斥了具有国家资质的企业

**应对措施**：
```
替代方案：
1. 删除地方证书要求
2. 改为国家、行业通用资质要求
3. 认可国外等同资质
```

#### 指标 2.6：行业组织成员身份要求风险 [A 级]
**规范要求**：不得要求企业取得特定行业组织成员身份

**风险表现**：
- 规定投标企业必须是某行业协会会员
- 要求企业加入商会才能投标
- 以会员身份作为投标资格条件

**检查清单**：
- [ ] 是否存在行业组织成员身份要求
- [ ] 该要求是否具有歧视性
- [ ] 是否与项目需求无关

**应对措施**：
```
清除措施：
1. 删除成员身份要求
2. 改为相关资质条件
3. 确保透明公正的遴选标准
```

#### 指标 2.7：其他不合理参投限制风险 [B 级]
**规范要求**：不得用其他不合理条件限制经营主体参投

**风险表现**：
- 设置的资格条件与项目需求明显不符
- 要求企业具备不必要的条件
- 变相设置进入壁垒

**检查清单**：
- [ ] 所有资格条件是否均为项目必需
- [ ] 条件是否客观公正
- [ ] 是否存在变相歧视

**应对措施**：
```
完整审查：
1. 列出所有资格条件
2. 逐一分析其必要性
3. 删除不必要条件
4. 保证公平竞争
```

### 模块三：标准招标文件风险

#### 指标 3.1：业绩区域差异性得分风险 [A 级]
**规范要求**：不得根据业绩地区设置差异性得分

**风险表现**：
- 招标文件规定本地业绩和外地业绩评分不同
- 为本地企业的业绩加分
- 对外地企业的同等业绩评分较低

**检查清单**：
- [ ] 招标文件评标标准中是否涉及业绩加分
- [ ] 业绩评分是否因地区而不同
- [ ] 权重设置是否合理

**应对措施**：
```
修订标准：
将"近X年在本地完成的类似项目..."
改为"近X年完成的国内同类项目..."
```

#### 指标 3.2：所有制形式差异性得分风险 [A 级]
**规范要求**：不得根据所有制形式设置差异性得分

**风险表现**：
- 国有企业和民营企业评分不同
- 内资企业和外资企业评分不同
- 大型央企与小企业评分不同

**检查清单**：
- [ ] 评标标准中是否涉及所有制背景
- [ ] 是否存在"国有企业额外加分"的规定
- [ ] 所有制形式是否影响评分

**应对措施**：
```
规范做法：
删除所有涉及所有制形式的评分条款
统一评标标准不区分所有制
```

#### 指标 3.3：产地差异性得分风险 [A 级]
**规范要求**：不得根据产品产地设置差异性得分

**风险表现**：
- 本地产品和进口产品评分不同
- 国产和进口产品质量等级认定不同
- 为采购本地产品的企业加分

**检查清单**：
- [ ] 招标条款中是否涉及产地要求或加分
- [ ] 产品评价标准是否因产地而不同
- [ ] 评分中是否隐含地歧视进口产品

**应对措施**：
```
改进方向：
以产品质量、价格、性能为评分标准，
不涉及产地因素
```

#### 指标 3.4：企业规模注册地址等差异性得分风险 [A 级]
**规范要求**：不得根据企业规模、注册地址等设置差异性得分

**风险表现**：
- 大型企业和小微企业评分不同
- 注册地在本市和外市评分不同
- 注册资金多和少评分不同
- 市场占有率高的企业加分
- 负债率低的企业加分

**检查清单**：
- [ ] 评标标准中是否包含企业规模评分
- [ ] 是否涉及注册地点、注册资金等条款
- [ ] 是否存在对大中小企业的差别对待
- [ ] 是否根据财务指标设置加分

**应对措施**：
```
完整纠正：
1. 删除所有规模类评分条款
2. 删除地址、资金相关条款
3. 删除财务状况评分（除必需的资质证明）
4. 确保中小企业的公平性
```

#### 指标 3.5：联合体成员身份差异性得分风险 [A 级]
**规范要求**：不得根据联合体成员身份设置差异性得分

**风险表现**：
- 全部本地成员联合体和本外地混合联合体评分不同
- 大企业联合和小企业联合评分不同
- 国有与民营联合评分不同

**检查清单**：
- [ ] 招标文件中是否涉及联合体成员评分
- [ ] 是否因成员身份设置差异评分
- [ ] 联合体形式是否受到不合理限制

**应对措施**：
```
修订方向：
删除涉及联合体成员身份的评分条款
统一评标标准不因成员差异而改变
```

#### 指标 3.6：其他排除或限制竞争内容风险 [B 级]
**规范要求**：不得设置其他排除或限制竞争的内容

**风险表现**：
- 隐形设置不合理的技术指标
- 过度规定品牌、型号等限制条件
- 其他变相排斥竞争的条款

**检查清单**：
- [ ] 招标文件条款是否均为项目必需
- [ ] 是否存在不合理的具体指定
- [ ] 条款是否可能排斥合格的竞争者

**应对措施**：
```
逐项审查：
列出所有限制性条款
分析其必要性和合理性
删除不必要限制
```

### 模块四：定标权保护风险

#### 指标 4.1：定标方法指定风险 [A 级]
**规范要求**：不得为招标人指定定标方法

**风险表现**：
- 政策要求必须采用某种定标方法
- 规定定标必须由评标委员会决定
- 强制招标人接受评委建议

**检查清单**：
- [ ] 定标权是否完全属于招标人
- [ ] 政策是否干涉定标方法选择
- [ ] 招标人是否有最终决定权

**应对措施**：
```
规范做法：
"定标权由招标人行使，
招标人可参考评标委员会的建议，
但定标决定由招标人独立作出"
```

#### 指标 4.2：定标单位或人员指定风险 [A 级]
**规范要求**：不得指定定标单位或人员

**风险表现**：
- 要求定标必须由评标委员会进行
- 规定定标权属于政府部门
- 禁止招标人单独决定定标

**检查清单**：
- [ ] 定标决定权归属是否明确
- [ ] 是否存在"必须由...定标"的规定
- [ ] 招标人定标权是否被保障

**应对措施**：
```
明确规定：
"定标权由招标人行使，
招标人为定标决定的责任人"
```

#### 指标 4.3：定标权转移风险 [A 级]
**规范要求**：不得将定标权交由招标人以外的其他单位行使

**风险表现**：
- 政府部门代替招标人进行定标
- 要求评标委员会直接定标
- 其他机构拥有定标决定权

**检查清单**：
- [ ] 定标权是否完全掌握在招标人手中
- [ ] 是否存在权力转移现象
- [ ] 其他部门是否参与定标决定

**应对措施**：
```
纠正措施：
1. 明确定标权属于招标人
2. 删除其他部门的定标权
3. 建立招标人责任制
```

#### 指标 4.4：抽签摇号强制定标风险 [A 级]
**规范要求**：不得规定直接以抽签等方式确定中标人

**风险表现**：
- 政策要求同分企业用抽签定标
- 在多个合格企业中规定用摇号确定中标
- 不允许招标人进行其他形式的选择

**检查清单**：
- [ ] 是否存在强制摇号规定
- [ ] 是否剥夺招标人的选择权
- [ ] 同分情况的处理是否合理

**应对措施**：
```
改进建议：
"出现同分情况时，招标人可采用
评分、重新谈判等方式进行比较，
必要时可采用随机抽取等方式"
```

#### 指标 4.5：其他定标权限制风险 [B 级]
**规范要求**：不得用其他不合理条件限制招标人定标权

**风险表现**：
- 要求定标时必须考虑的非技术因素
- 强制招标人在有分歧时上报政府
- 其他不合理的限制

**检查清单**：
- [ ] 定标过程中是否存在不合理约束
- [ ] 招标人的意愿是否被充分尊重
- [ ] 定标权是否有其他隐性限制

**应对措施**：
```
梳理定标全流程
删除不合理限制
保证招标人的最终决定权
```

### 模块五：信用评价应用风险

#### 指标 5.1：信用信息差异记录风险 [A 级]
**规范要求**：不得对不同地区或所有制形式作区别规定

**风险表现**：
- 本地企业和外地企业的失信记录标准不同
- 国有企业和民营企业的信用要求不同
- 不同所有制形式企业的信用信息归集要求不同

**检查清单**：
- [ ] 信用信息记录标准是否统一
- [ ] 是否存在地区差异规定
- [ ] 是否存在所有制差异规定

**应对措施**：
```
统一规范：
"对所有经营主体采用统一的
信用信息记录、归集、共享标准"
```

#### 指标 5.2：信用评价标准差异风险 [A 级]
**规范要求**：不得对不同地区或所有制形式采用不同信用评价标准

**风险表现**：
- 本地和外地企业的失信定义不同
- 不同所有制企业的信用评价权重不同
- 业绩评价对不同企业的标准不同

**检查清单**：
- [ ] 信用评价标准是否透明统一
- [ ] 是否根据地区或所有制制定不同标准
- [ ] 评价权重是否公平

**应对措施**：
```
规范做法：
"采用统一的信用评价标准，
对所有经营主体一视同仁"
```

#### 指标 5.3：差异化信用监管风险 [A 级]
**规范要求**：不得根据地区或所有制采取差异化监管

**风险表现**：
- 对本地企业的失信处罚较轻
- 对外地企业的监管更严格
- 不同所有制企业获得的信用待遇不同

**检查清单**：
- [ ] 信用监管是否一致
- [ ] 监管措施是否受地区影响
- [ ] 企业所有制是否影响监管

**应对措施**：
```
确保公平：
"信用监管对所有经营主体采用
相同的标准和程序"
```

#### 指标 5.4：信用结果使用权限制风险 [A 级]
**规范要求**：不得没有法定依据限制经营主体的自主权

**风险表现**：
- 政策规定投标企业必须使用信用评价结果
- 强制招标人参考某信用评价
- 禁止企业引用其他信用信息

**检查清单**：
- [ ] 信用评价结果的使用是否强制
- [ ] 企业是否有选择的自由
- [ ] 是否仅认可某个信用信息源

**应对措施**：
```
修订方向：
改为"可鼓励企业和招标人参考
国家及社会信用评价结果"
```

#### 指标 5.5：其他信用相关不公平措施风险 [B 级]
**规范要求**：不得制定其他排除限制竞争或损害合法权益的措施

**风险表现**：
- 信用信息公开不充分
- 企业的异议权利不完善
- 信用修复机制不健全

**检查清单**：
- [ ] 信用信息是否充分公开
- [ ] 企业是否有异议和修复途径
- [ ] 是否存在不合理的长期限制

**应对措施**：
```
完善措施：
1. 建立完善的信息公开机制
2. 确保企业的异议权和修复权
3. 明确失信人员解除期限
```

### 模块六：交易监管与服务风险

#### 指标 6.1：交易服务机构职能越权风险 [A 级]
**规范要求**：不得规定交易服务机构行使行政管理职能

**风险表现**：
- 公共资源交易平台对项目进行审批
- 要求交易机构对招标文件备案
- 交易机构对投标企业进行资格监管
- 交易机构行使罚款权

**检查清单**：
- [ ] 公共资源交易平台机构是什么性质
- [ ] 是否拥有审批权、监管权、处罚权
- [ ] 是否超越了服务机构的权限

**应对措施**：
```
纠正措施：
1. 明确交易平台为服务性机构
2. 删除行政管理职能
3. 行政权力由政府部门行使
4. 交易平台只提供场所和技术支持
```

#### 指标 6.2：非公共资源强制进入平台风险 [A 级]
**规范要求**：不得强制非公共资源项目进入公共资源交易平台

**风险表现**：
- 政府部门要求所有招标项目进入平台
- 社会资本项目必须在平台交易
- 违反法律规定的项目也被强制纳入

**检查清单**：
- [ ] 是否存在强制进入平台的规定
- [ ] 项目范围是否明确且合法
- [ ] 企业是否有选择的权利

**应对措施**：
```
澄清规定：
"法律规定必须招标的项目进入平台，
其他项目可自主选择"
```

#### 指标 6.3：告知承诺原件强制提交风险 [A 级]
**规范要求**：不得强制提供能通过告知承诺和核验的原件

**风险表现**：
- 要求投标企业在投标时提供营业执照原件
- 强制提供资质证书原件
- 不接受复印件和电子版本

**检查清单**：
- [ ] 是否存在强制提交原件的要求
- [ ] 该信息是否可通过事后核验获得
- [ ] 是否可以接受承诺和异常情况处理

**应对措施**：
```
改进做法：
"投标人可提供复印件，
招标人在中标后进行原件核验"
```

#### 指标 6.4：特定人员到场要求风险 [A 级]
**规范要求**：不得违法要求法定代表人等到场

**风险表现**：
- 开标时要求投标企业法定代表人必须到场
- 要求技术负责人或项目经理参加开标会
- 开标现场有特定人员才能投标

**检查清单**：
- [ ] 开标规则中是否有人员出席要求
- [ ] 是否对投标企业造成不合理负担
- [ ] 法律是否规定必须出席

**应对措施**：
```
改进规定：
"投标企业可派代理人参加开标，
法定代表人出席与否由企业自主决定"
```

#### 指标 6.5：其他不当限制措施风险 [B 级]
**规范要求**：不得制定其他不当限制企业参与的措施

**风险表现**：
- 对企业在平台注册的不合理限制
- 收取过高的交易服务费用
- 其他隐形的参与障碍

**检查清单**：
- [ ] 注册程序是否简洁合理
- [ ] 服务收费是否公正
- [ ] 是否存在其他进入壁垒

**应对措施**：
```
完整检查：
1. 简化注册程序
2. 规范服务费用
3. 删除不合理限制
4. 建立救济渠道
```

### 模块七：保证金相关风险

#### 指标 7.1：保证金收取限制风险 [A 级]
**规范要求**：不得限制招标人依法收取保证金

**风险表现**：
- 政策禁止招标人收取投标保证金
- 规定不能收取履约保证金
- 虽然法律允许但政策层面限制收取

**检查清单**：
- [ ] 招标人收取保证金的权利是否被限制
- [ ] 政策是否与法律规定相悖
- [ ] 招标人的自主权是否受限

**应对措施**：
```
删除限制条款
"招标人可根据项目实际需要
依法收取投标保证金、履约保证金等"
```

#### 指标 7.2：法定外保证金要求风险 [A 级]
**规范要求**：不得要求除法定外的其他保证金

**风险表现**：
- 政策要求缴纳"安全生产保证金"
- 规定缴纳"质量风险金"
- 其他法律未规定的保证金

**检查清单**：
- [ ] 保证金种类有哪些
- [ ] 是否均为法律规定的四种
  - 投标保证金
  - 履约保证金
  - 工程质量保证金
  - 农民工工资保证金
- [ ] 是否存在其他保证金要求

**应对措施**：
```
清单纠正：
列出所有保证金种类
删除四种法定外的保证金
```

#### 指标 7.3：保证金形式限定风险 [A 级]
**规范要求**：不得限定保证金形式

**风险表现**：
- 只接受现金，不接受支票或汇款
- 只接受银行保函，不接受保险保函
- 要求通过特定银行转账

**检查清单**：
- [ ] 招标文件中保证金形式是否单一
- [ ] 投标企业是否有多种选择
- [ ] 是否可以选择现金、保函、担保等

**应对措施**：
```
扩大选择：
"保证金可以现金、银行保函、
保险保函等形式提供"
```

#### 指标 7.4：保函出具机构指定风险 [A 级]
**规范要求**：不得要求从特定机构开具保函

**风险表现**：
- 只能从指定银行开具保函
- 必须由政府指定的担保公司担保
- 要求从特定保险公司购买保险

**检查清单**：
- [ ] 是否指定了保函出具机构
- [ ] 企业是否有多家机构可选
- [ ] 是否形成了垄断

**应对措施**：
```
开放选择：
删除特定机构要求
改为"由符合资质的金融机构出具"
```

#### 指标 7.5：保证金退还前置条件风险 [A 级]
**规范要求**：不得在招标文件外设定退还前置条件

**风险表现**：
- 招标文件说退中标后退，但实际要求完成工程后退
- 设定额外的退还条件如获奖项等
- 退还程序不明确或过于复杂

**检查清单**：
- [ ] 保证金退还条件是否在招标文件中明确
- [ ] 是否存在招标文件外的额外条件
- [ ] 退还程序是否清晰高效

**应对措施**：
```
规范退还：
在招标文件中明确详细的退还条件和程序
不得增加招标文件外的条件
```

#### 指标 7.6：其他保证金不合理限制风险 [B 级]
**规范要求**：不得制定其他不合理限制

**风险表现**：
- 保证金数额过高
- 退还时间过长
- 保证金挪作他用

**检查清单**：
- [ ] 保证金数额是否合理
- [ ] 是否按规定及时退还
- [ ] 是否专款专用

**应对措施**：
```
完整规范：
1. 合理确定保证金数额
2. 明确退还期限
3. 建立保证金管理制度
4. 接受监督
```

第一步：风险识别

- [ ] 对标模块一（招标人自主权）的 11 项指标

**阶段 4：标准文件审查**
- [ ] 对标模块三（标准招标文件）的 6 项指标

- [ ] 对标模块四（定标权）的 5 项指标

## 第四部分：风险应对工作机制

### 1. 日常监测机制

**监测内容**：
- 新制定的招标投标相关政策
- 招标文件中的异常条款
- 社会投诉举报情况
- 行业协会反映的问题

**监测频率**：
- 政策制定部门提前告知：政策制定时同步审查
- 社会投诉：及时响应处理
- 定期检查：每季度一次

**监测责任部门**：
- 本级政府部门
- 招标投标行政监督部门
- 公共资源交易中心

### 2. 整改落实机制

**整改步骤**：
1. **问题确认**：确认是否存在违规
2. **原因分析**：分析违规原因
3. **方案制定**：制定详细整改方案
4. **组织实施**：按期完成整改
5. **验收检查**：确认整改有效
6. **记录存档**：保存整改记录

**整改期限**：
- A 级风险：30 日内完成
- B 级风险：60 日内完成
- C 级风险：90 日内完成
- D 级风险：按实际需要安排

### 3. 监督管理机制

**监督主体**：
- 上级政府部门
- 公平竞争审查部门
- 社会监督（企业、中介、公众）

**监督方式**：
- 定期评估：每年组织政策措施评估
- 投诉举报处理：建立线索征集渠道
- 社会评议：听取利益相关方意见
- 第三方评估：委托专业机构评估

**处理措施**：
- 发现违规即时纠正
- 拒不改正将面临处分
- 相关责任人将依法追究

### 4. 信息公开机制

**公开内容**：
- 公平竞争审查工作机制
- 审查标准和流程
- 政策措施审查结论

## 第五部分：典型案例与启示

### 案例一：招标代理机构指定的问题

**案例描述**：
某市政府出台文件规定，本市范围内的政府购买服务项目，必须选择市级招标代理协会推荐的招标代理机构。

**违规分析**：
- 限制了招标人对招标代理机构的自主选择
- 违反公平竞争审查规则第五条第一项
- 可能垄断招标代理市场

**整改建议**：
删除强制性规定，改为：
"招标人可自主选择具有相应资质的招标代理机构"

---

### 案例二：本地业绩要求问题

**案例描述**：
某市招标文件标准模板规定，施工企业必须具有本地区三年以上的施工业绩，否则不符合资格条件。

**违规分析**：
- 设置了本地业绩的资格限制
- 造成外地企业无法参加投标
- 违反了全国统一市场准入的原则
- 违反公平竞争审查规则第六条第三项

**整改建议**：
修改为：
"施工企业应具有相应资质和三年以上同类工程施工业绩，不限定业绩所在地"

---

### 案例三：评标权重的差异性问题

**案例描述**：
某市的招标文件示范文本中规定，投标人为本地企业的加 3 分，为本地国有企业的加 5 分。

**违规分析**：
- 根据所有制形式设置差异性得分
- 造成企业间不公平竞争
- 违反公平竞争审查规则第七条第二项

**整改建议**：
删除所有制相关加分，统一评标标准：
"评标委员会按照招标文件规定的评标标准和方法进行评标，不因企业所有制形式而改变"

---

### 案例四：强制进入交易平台问题

**案例描述**：
某市政府规定，社会资本组织的招标项目也必须在本市公共资源交易平台交易，否则不承认中标结果。

**违规分析**：
- 超出了法律规定的强制招标范围
- 限制了非公共资源项目的交易自由
- 违反公平竞争审查规则第十条第二项

**整改建议**：
修改为：
"法律规定必须招标的项目应进入公共资源交易平台，其他项目由招标人自主选择"

### 附录 2：常见问题解答

**Q1：政府能否在招标文件中推荐某个招标代理机构？**
A：可以提供示范文本或推荐，但不得强制指定。招标人仍有自由选择权。

**Q2：是否可以要求投标人提供本地的营业执照副本？**
A：可以作为真实性验证，但不能将本地注册作为资格条件，外地企业的异地营业执照应同等认可。

**Q3：标准招标文件中能否规定评标委员会必须按照权重给分？**
A：可以规定权重，但权重的设置不能违反公平竞争规则，不能根据地区、所有制等设置差异。

**Q4：能否规定中小企业在招标中必须获胜？**
A：不能。公平竞争审查规则要求对所有企业一视同仁，不能按规模差异对待。

**Q5：如何处理已经出台的违规政策措施？**
A：应立即纠正，发布修订文件，向相关企业澄清，并保存整改记录。
