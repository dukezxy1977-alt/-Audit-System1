# 招标审计规则库 - 使用指南

## 项目概述

本项目根据 《招标投标领域公平竞争审查规则》（2024 年 5 月 1 日施行）建立的完整的招投标行为风险规则库。

**项目来源**：
- 文件 1：P020240403302339643235.pdf - 招标投标领域公平竞争审查规则（国家发改委）
- 文件 2：W020240511360535151619.pdf - 相关政策补充文件

**项目目标**：
- 帮助政府部门规范招标投标行为
- 防范公平竞争审查中的违规风险
- 建立健全市场准入和竞争环境
- 保护经营主体的合法权益

## 关键概念解释

### 什么是公平竞争审查？

> 政策制定机关对拟制定的招标投标领域政策措施是否存在排除、限制竞争情形进行的审查评估活动。

**简单说**：检查政策是否公平对待所有企业。

### 什么是不合理的市场壁垒？

常见表现：
- **地域壁垒**：优待本地企业，排斥外地企业
- **所有制壁垒**：优待国有企业，歧视民营企业
- **规模壁垒**：优待大企业，歧视中小企业
- **身份壁垒**：要求特定的协会会员、政府官员推荐等

### 政策措施什么时候必须进行审查？

```
所有拟制定的招标投标领域政策措施
都必须在提请审议或报批前完成审查
   ↓
一旦发现问题，改正后再出台
或改不了就不出台
```

### 谁来做审查？

**责任人**：政策制定机关
**具体做**：指定的专门机构
**审查内容**：对标本规则的各项要求
**输出**：书面的审查结论

## 常见错误及改正示范

### 错误 1：指定招标代理机构

❌ **错误表述**：
```
"本市范围内的政府项目，必须通过市招标协会推荐的
代理机构进行招标。"
```

✓ **正确表述**：
```
"招标人可自主选择具有相应资质的招标代理机构，
市招标协会维持的代理机构名录仅供参考。"
```

---

### 错误 2：要求本地业绩

❌ **错误表述**：
```
"施工企业应具有三年以上本市类似工程施工业绩。"
```

✓ **正确表述**：
```
"施工企业应具有三年以上国内同类工程施工业绩。"
```

---

### 错误 3：设置所有制差异加分

❌ **错误表述**：
```
招标文件评分标准：
- 国有企业或国企控股企业：加 5 分
- 其他企业：加 0 分
```

✓ **正确表述**：
```
招标文件评分标准：
- 投标企业的财务状况、技术能力、管理团队等
（不涉及所有制因素）
```

---

### 错误 4：强制进入交易平台

❌ **错误表述**：
```
"所有招标项目必须在市公共资源交易平台进行，
违者不承认中标结果。"
```

✓ **正确表述**：
```
"法律规定的必须招标的项目应进入公共资源交易平台。
其他项目由招标人自主决定。"
```

## 风险级别说明

### A 级 - 明确违法（需立即纠正）

**特点**：
- 直接违反法律规定
- 属于排斥、限制竞争的明显情形
- 可能引发企业投诉或行政诉讼

**处理**：
- 发现即改：立即取消违规条款
- 公开纠正：发布修订文件或澄清说明
- 责任追究：记录违规人员和纠正过程

**案例**：指定招标代理机构、要求本地分支机构、强制指定定标方法等

### B 级 - 原则性错误（需限期改正）

**特点**：
- 违反了公平竞争的基本原则
- 可能导致不公平对待的情形
- 容易被投诉但规定不如 A 级明确

**处理**：
- 制定计划：在规定时间内完成改正
- 逐步完善：优化相关规定和流程
- 监督检查：定期评估改正效果

**案例**：权重设置欠合理、程序规定不清等

### C 级 - 中等风险（需重点监测）

**特点**：
- 存在不够规范的地方
- 可能产生市场壁垒的隐患
- 需要通过完善制度来消除

**处理**：
- 重点监测：在日常工作中观察
- 完善机制：改进相关规定
- 定期审查：每季度重新评估

**案例**：信息公开不充分、投诉救济不完善等

### D 级 - 低风险（改进完善）

**特点**：
- 程序性或技术性缺陷
- 不直接涉及竞争限制
- 需要优化但不紧迫

**处理**：
- 制定计划：根据实际情况安排
- 逐步改进：在相关工作中完善
- 定期总结：建立长效机制

**案例**：文件格式不规范、索引不完善等
