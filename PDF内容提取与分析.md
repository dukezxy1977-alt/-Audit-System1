## 二、第一份文件详细内容

### 文件：招标投标领域公平竞争审查规则

#### 第一章 总则

**核心内容：**
- 根据《招标投标法》《招标投标法实施条例》制定
- 公平竞争审查是指行政机关和具有管理公共事务职能的组织对拟制定的招标投标领域政策措施是否存在排除、限制竞争情形进行审查评估

**关键规定：**
1. 未经公平竞争审查或审查存在违规的政策不得出台
2. 政策制定机关应指定专门机构负责审查工作
3. 多部门联合制定时，由牵头部门组织开展审查

#### 第二章 审查标准

**第五条：招标人自主权保护（11 项禁止）**
不得制定以下政策：
1. 为招标人指定招标代理机构
2. 为招标人指定投标资格、技术、商务条件
3. 为招标人指定资格审查方法或评标方法
4. 为招标人指定资格审查标准或评标标准
5. 为招标人指定评标委员会成员
6. 限制招标人选择电子交易系统
7. 强制选择电子认证服务
8. 为招标人指定特定交易工具
9. 为招标人指定承包商预选库
10. 要求招标人依照地方扶持政策进行招标
11. 其他不合理限制招标人自主权

**第六条：市场准入条件统一（6 项禁止）**
经营主体参投资格不得制定：
1. 在市场准入负面清单以外要求取得行政许可
2. 要求在本地区设立分支机构、缴纳税收社保
3. 要求取得本地区业绩或奖项
4. 要求取得特定地区或行业证书
5. 要求取得特定行业组织成员身份
6. 其他不合理限制

**第七条：标准招标文件制定（6 项禁止）**
不得设置以下内容：
1. 根据业绩区域设置差异性得分
2. 根据所有制形式设置差异性得分
3. 根据产品产地设置差异性得分
4. 根据企业规模、注册地址等设置差异性得分
5. 根据联合体成员注册地址设置差异性得分
6. 其他排除或限制竞争内容

**第八条：定标权保护（5 项禁止）**
不得制定以下政策：
1. 为招标人指定定标方法
2. 为招标人指定定标单位或人员
3. 将定标权交由他人行使
4. 规定用抽签、摇号等确定中标人
5. 其他限制定标权

**第九条：信用评价（5 项禁止）**
1. 对不同地区或所有制形式作区别规定
2. 采用不同信用评价标准
3. 采取差异化信用监管
4. 限制经营主体参考使用信用评价结果
5. 其他排除限制竞争

**第十条：交易监管（5 项禁止）**
1. 规定交易服务机构行使行政管理职能
2. 强制非公共资源项目进入公共资源平台
3. 强制提供原件
4. 违法要求特定人员到场
5. 其他不当限制

**第十一条：保证金（6 项禁止）**
1. 限制招标人依法收取保证金
2. 要求缴纳除法定保证金外的其他保证金
3. 限定保证金形式
4. 要求从特定机构开具保函
5. 在招标文件外设定前置条件
6. 其他不合理限制

#### 第三章 审查机制

1. **建立审查工作机制**：明确负责机构、审查标准、审查流程
2. **完成时限**：在提请审议或报批前完成审查
3. **审查结论**：作出符合或不符合的书面结论
4. **意见征求**：听取有关经营主体、行业协会意见；向社会公开征求
5. **第三方评估**：可委托评估公平竞争影响和竞争效果

#### 第四章 监督管理

1. **定期评估**：地方招标投标部门定期评估政策措施
2. **利益反映**：公民、法人可向政策制定机关反映
3. **线索征集**：建立市场壁垒线索征集机制
4. **异议投诉**：规范处理资格预审文件、招标文件异议和投诉
5. **责任追究**：未进行公平竞争审查或违反标准的机构及责任人将面临处分

#### 第五章 附则

1. 政策制定机关作为招标人、公共资源平台应参照本规则开展审查
2. 本规则由国家发展改革委负责解释
3. 自 2024 年 5 月 1 日起施行

## 三、主要风险点汇总

### 高风险行为（违反法规）
- ✗ 为招标人指定招标代理机构、评标方法、中标人等
- ✗ 设置地域、所有制形式、产地等差异性要求
- ✗ 强制企业在本地区设分支机构或缴纳税社保
- ✗ 限制企业市场准入或竞争参与
- ✗ 不公开进行公平竞争审查

### 中等风险行为（需重点关注）
- ⚠ 行使不属于自己的定标权
- ⚠ 规定不合理的资格条件
- ⚠ 强制使用特定交易工具
- ⚠ 过度限制保证金形式

### 低风险行为（规范化需求）
- ◐ 信息公开不充分
- ◐ 意见征求程序不完善
- ◐ 审查记录不完整
